## Description
This repository contains solutions to 4 basic programming problems provided as part of the TANDEMLOOP Stage-1 Screening Test.  
All problems are solved using JavaScript and executed using Node.js.

## Files Included
- Problem-1.js – Calculator using class (Addition, Subtraction, Multiplication, Division)
- Problem-2.js – Generate odd number series based on input
- Problem-3.js – Conditional odd number series based on input
- Problem-4.js – Count of numbers that are multiples of digits 1 to 9

## How to Run the Programs
Make sure Node.js is installed on your system.

Run each file using the following command:
```bash
node Problem-1.js
node Problem-2.js
node Problem-3.js
node Problem-4.js
